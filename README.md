**Stock Market Portfolio Simulator & Visualizer** is a comprehensive financial analytics tool built with Python that simulates multi-asset stock portfolio performance. This project demonstrates practical applications of data science libraries including Pandas, NumPy, and Matplotlib to analyze investment returns, calculate risk metrics, and visualize portfolio growth over time.

The simulator allows users to track daily returns of multiple stocks (Apple, Google, Bitcoin), calculate cumulative portfolio growth, and generate professional financial visualizations. This project is perfect for understanding how quantitative finance and portfolio optimization work in the real world.

### PROBLEM SOLVED

| PROBLEM | SOLUTION PROVIDED |
|---------|-------------------|
| Manual portfolio tracking is time-consuming | Automated daily return calculations |
| No clear understanding of investment growth | Cumulative return tracking system |
| Difficulty visualizing portfolio performance | Professional Matplotlib charts |
| Complex financial math for beginners | Step-by-step simplified calculations |
| No risk assessment metrics | Standard deviation for volatility tracking |

06 STEP BY STEP CODE EXPLANATION
STEP 1: IMPORT LIBRARIES
python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
EXPLANATION: Python ma default ma dherai data science tools hudainan. Hami le baira bata teen main toolboxes bolayeko chham:

numpy lai np name diyeko - yo numbers ra matrices ko lagi

pandas lai pd name diyeko - yo Excel jasto table data handle garna

matplotlib.pyplot lai plt name diyeko - yo graph ra charts draw garna

STEP 2: CREATE DATE RANGE
python
dates = pd.date_range(start="2026-05-01", periods=10)
EXPLANATION: Stock market maa price tracking ko lagi dates chahincha. pd.date_range() le automatic sequential dates banaucha:

start="2026-05-01" - suru date fix garyo

periods=10 - 10 din samma ko dates create garyo

OUTPUT: ['2026-05-01', '2026-05-02', '2026-05-03', ... '2026-05-10']

STEP 3: CREATE SIMULATED STOCK PRICES
python
apple_prices = [150, 152, 151, 155, 154, 158, 157, 160, 162, 165]
google_prices = [2800, 2810, 2795, 2820, 2830, 2815, 2840, 2860, 2850, 2880]
bitcoin_prices = [60000, 61000, 59500, 62000, 61500, 63000, 62500, 64000, 63800, 65000]
EXPLANATION: Real stock API nabhayeko le manual data create gareko:

Apple ko price: 150 bata suru vai 165 samma pugcha

Google ko price: 2800 bata 2880 samma

Bitcoin ko price: 60000 bata 65000 samma

Square brackets [] le Python list bancha. Harek list maa exactly 10 items chan, kina ki mathi 10 din ko dates banayeko chham.

STEP 4: CREATE PANDAS DATAFRAME
python
data = {
    'Apple': apple_prices,
    'Google': google_prices,
    'Bitcoin': bitcoin_prices
}
df = pd.DataFrame(data, index=dates)
EXPLANATION: Curly brackets {} le Python dictionary bancha. Dictionary maa:

Key: Column name (Apple, Google, Bitcoin)

Value: Tyo stock ko price list

pd.DataFrame() le dictionary lai table maa convert garchha. index=dates le rows maa dates rakhnu bhanyo.

RESULTING TABLE:

Date	Apple	Google	Bitcoin
2026-05-01	150	2800	60000
2026-05-02	152	2810	61000
2026-05-03	151	2795	59500
...	...	...	...
STEP 5: CALCULATE DAILY RETURNS
python
daily_returns = df.pct_change()
daily_returns = daily_returns.dropna()
EXPLANATION: .pct_change() le hijo ko price ra aaja ko price compare garera percentage change nikalcha.

FORMULA: (Aaja ko price - Hijo ko price) / Hijo ko price

Day 1: No previous data → NaN (empty)

Day 2: Apple (152-150)/150 = 0.0133 (1.33% profit)

Day 3: Apple (151-152)/152 = -0.0066 (0.66% loss)

.dropna() le NaN values (Day 1) lai hataucha.

STEP 6: SET PORTFOLIO WEIGHTS
python
weights = np.array([0.4, 0.3, 0.3])
EXPLANATION: Hami sanga total 100% money cha:

40% (0.4) Apple maa laganyo

30% (0.3) Google maa laganyo

30% (0.3) Bitcoin maa laganyo

np.array() le normal list lai NumPy array maa convert garchha, jasle mathematical calculations fast hunchha.

STEP 7: CALCULATE PORTFOLIO DAILY RETURNS
python
portfolio_daily_return = daily_returns.dot(weights)
EXPLANATION: .dot() le matrix multiplication garchha.

FORMULA: (Apple Return × 0.4) + (Google Return × 0.3) + (Bitcoin Return × 0.3)

EXAMPLE: Day 2 maa:

Apple return: 0.0133 × 0.4 = 0.00532

Google return: 0.0036 × 0.3 = 0.00108

Bitcoin return: 0.0167 × 0.3 = 0.00501

Total Portfolio Return = 0.01141 (1.14% profit)

STEP 8: CALCULATE CUMULATIVE RETURN
python
portfolio_cumulative_return = (1 + portfolio_daily_return).cumprod() - 1
EXPLANATION: Compound interest jasto, din din ko growth accumulate garne.

1 + portfolio_daily_return: Each day ko return maa 1 jodne (e.g., 0.01 + 1 = 1.01)

.cumprod(): Harek din ko value lai multiply gardai janey

- 1: Last maa 1 ghatayera actual profit percentage nikalney

EXAMPLE:

Day 2: 1.0114

Day 3: 1.0114 × 0.9985 = 1.0099

Day 4: 1.0099 × 1.015 = 1.025

After 10 days: 1.075 → 7.5% total profit

STEP 9: CREATE VISUALIZATION
python
plt.figure(figsize=(10, 5))
plt.plot(portfolio_cumulative_return, marker='o', color='purple', 
         label='Total Portfolio Return')
plt.title('Stock Portfolio Growth Over 10 Days')
plt.xlabel('Dates')
plt.ylabel('Cumulative Profit Percentage')
plt.grid(True)
plt.legend()
plt.show()
EXPLANATION:

plt.figure(figsize=(10, 5)): Graph canvas banayo (10 inch wide, 5 inch tall)

plt.plot(): Line chart draw garyo

marker='o': Harek point maa circle marker rakhyo

color='purple': Line purple color maa banayo

label: Legend ko lagi name diyo

plt.title(): Graph ko heading

plt.xlabel(): X-axis (tala) maa label

plt.ylabel(): Y-axis (left) maa label

plt.grid(True): Background maa grid lines rakhyo

plt.legend(): Color code ko box dekhaayo

plt.show(): Graph screen maa display garyo
